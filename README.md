## SkillFactory_RealDS_Mod_1.1
### Курс SkillFactory. Модуль "О вкусной и здоровой пище"

Модуль посвящен:
1. подготовке данных о ресторанах на основании имеющегося датафрейма
2. обучению модели RandomForestRegressor на подготовленных данных
3. прогнозирование рейтинга ресторанов с помощью модели, обученной на предыдущем шаге
